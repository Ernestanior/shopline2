# 图片目录说明

## 📁 目录结构

```
images/
├── products/       # 商品图片
│   ├── product-1.jpg
│   ├── product-2.jpg
│   └── ...
├── hero/          # 首页横幅图片
│   ├── banner-1.jpg
│   └── ...
└── categories/    # 分类图片
    ├── category-1.jpg
    └── ...
```

## 📸 图片规格

### 商品图片 (products/)
- **尺寸**: 800x1200px (2:3 比例)
- **格式**: JPG
- **大小**: < 200KB (压缩后)
- **命名**: product-{id}.jpg

### 横幅图片 (hero/)
- **尺寸**: 1920x600px
- **格式**: JPG
- **大小**: < 300KB
- **命名**: banner-{id}.jpg

### 分类图片 (categories/)
- **尺寸**: 600x600px (1:1 比例)
- **格式**: JPG
- **大小**: < 150KB
- **命名**: category-{name}.jpg

## 🎯 获取图片

请参考项目根目录的 `IMAGE_GUIDE.md` 文件，了解如何从免费图库下载高质量图片。

推荐图库：
- Unsplash: https://unsplash.com
- Pexels: https://pexels.com
- Pixabay: https://pixabay.com

## 🔧 图片优化

下载图片后，建议使用以下工具压缩：
- TinyPNG: https://tinypng.com
- Squoosh: https://squoosh.app

## ⚠️ 注意事项

1. 确保图片允许商业使用
2. 保持统一的宽高比
3. 压缩图片以提高加载速度
4. 使用有意义的文件名
