1. **緊迫感** - 庫存警告、倒計時、限時優惠
2. **社交證明** - 實時購買、顧客評價、瀏覽人數
3. **視覺層級** - 清晰的信息架構
4. **互動反饋** - 懸停效果、點擊動畫
5. **信任建立** - 服務保證、退換貨政策
6. **降低摩擦** - 快速查看、一鍵加購

### 技術棧

- **框架**: Next.js 15 (App Router)
- **語言**: TypeScript
- **樣式**: Tailwind CSS + Custom CSS
- **圖片**: Next.js Image
- **狀態**: React Hooks
- **動畫**: CSS Animations + Transitions
- **後端**: Cloudflare Workers + D1
- **ORM**: Drizzle
- **API**: Hono Framework
